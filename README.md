# Retail Sales Analysis (Python + SQL)

## Overview
Exploratory Data Analysis on Sample Superstore dataset to understand sales, profit, and business insights.

## Tools & Technologies
- Python (pandas, matplotlib, seaborn)
- SQL (SQLite)
- Jupyter Notebook

## Key Insights
- West region leads in sales but Central has lowest margins.
- Technology is highly profitable; Furniture suffers due to heavy discounts.
- High discounts (>30%) often lead to losses.
- Strong seasonal trend with peaks in Q4.

## How to Run
1. Clone/download the repository
2. Place `Superstore sales dataset.csv` in `data/` folder
3. Run `python -m jupyter notebook`
4. Open `analysis.ipynb`
